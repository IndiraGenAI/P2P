
# ERP DB package

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


## Installation

```bash
$ yarn
```

## Build Package

```bash
$ yarn build
```

## Generate Package

```bash
# Generate package
$ yarn pack
# Build and generate package
$ yarn build:pack
```

## Deploy Package to micro service

```bash
# Deploy Package
$ yarn deploy
# Build and generate package and deploy package
$ yarn build:pack:deploy
```

## Entity Generate

```bash
# Generate Entity of typeorm
$ yarn db
```
