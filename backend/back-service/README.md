
# ERP Backend - Nestjs

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Set env

Rename `.evn.example` to `.env`


## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn build
$ yarn start:prod
```

## Code Manage

```bash
# Code format using prettier
$ yarn format

# Code validation using ESLint
$ yarn lint

```

## Swagger

It will be opened on [http://localhost:3010/swagger](http://localhost:3010/swagger).
It can be accessed on DEV mode only.
